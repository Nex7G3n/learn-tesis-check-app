# Archivos de referencia para la revisión con IA

Coloca aquí los siguientes archivos que mencionaste:

1. **PROMPT.docx** → Contiene las instrucciones/detalle de cómo debe revisar Gemini la tesis.
2. **EsquemaPT.docx** → Contiene la estructura y normas que debe verificar.

## Formato

Estos archivos deben estar en formato **.docx** (Word). El backend los leerá automáticamente al momento de analizar una tesis.

## Nota importante

- Los nombres de archivo deben coincidir exactamente: `PROMPT.docx` y `EsquemaPT.docx`.
- Si cambias el contenido, solo reemplaza los archivos y reinicia el servidor; no es necesario modificar código.
